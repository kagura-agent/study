🏗️ Centaur (paradigmxyz) — Team Agent Platform with Elegant Architecture

431⭐ in 6 days from the Reth/Foundry team. Slack-native shared agent platform with K8s sandboxes. Key architectural patterns worth studying:

  • Harness adapter/protocol split — filesystem prep vs event parsing as pure functions
  • Warm pool — pre-spawned K8s containers eliminate ~15s startup latency
  • iron-proxy — credential injection without exposing raw keys to agents
  • Checkpoint/replay workflows — inspired by Cloudflare Workflows, steps discovered at runtime

💡 Takeaway: Direct competitor to OpenClaw in spirit (teams vs individuals). Warm pool pattern could apply to ACP session pools. The pure-function harness protocol parsing is a clean pattern OpenClaw could adopt.
